*

Every minute

0-59

0 is the first minute of the hour.
-

Ranges are two numbers separated by hyphen. Ranges are inclusive. The first number must be less or equal to the second one.
Range ex: 4-6
~

We can randomize execution in a range with ~.
Range ex: 4~6
,

Lists are numbers separated by commas.
Example: 3,5,7

On many systems we can use ranges in a list
Example: 3-5,7-9

Ranges in list will not work on BSD cron. There we can only use list or range.
/

Step can be used with ranges or *. Adding / and a number after a range will step over minutes. Ranges with step are defined first-last/step. It will run at first then step too next until it reaches the last one.
Example 1-10/2 Will run 1,3,5,7,9

None Standard Options

The `@' commands that appear in place of the first five fields are extensions.

Value Explanation
@reboot Run once, at startup
@yearly Run once a year, "0 0 1 1 *"
@annually (same as @yearly)
@monthly Run once a month, "0 0 1 * *"
@weekly Run once a week, "0 0 * * 0"
@daily Run once a day, "0 0 * * *"
@midnight (same as @daily)
@hourly Run once an hour, "0 * * * *"
