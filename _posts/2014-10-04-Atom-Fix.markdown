Started using [Atom][atom] on Linux a little bit. Oddly it seems to break from time to time. When trying to open files it will give a nasty error. This is a quik fix.

{% highlight bash %}
cd ~/.atom/storage
find . -type f -delete
{% endhighlight %}

[atom]:        https://atom.io
